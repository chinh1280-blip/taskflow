Kết nối Supabase
Nhập thông tin Supabase để đồng bộ dữ liệu trên mọi thiết bị.
1
Vào supabase.com → Tạo project mới → chọn region Singapore
2
Vào SQL Editor → chạy script tạo bảng (xem hướng dẫn kèm theo)
3
Vào Settings → API → copy Project URL và anon key
Thông tin được lưu trên thiết bị này. Miễn phí vĩnh viễn cho dùng cá nhân.
